## ⚙️ File 4 – `.github/workflows/run-scraper.yml`

This workflow lets you run the scraper automatically in the cloud through **GitHub Actions**.

```yaml
name: Run Bright Data Scraper

on:
  workflow_dispatch:     # allows manual runs
  schedule:
    - cron: "0 */6 * * *"  # every 6 hours

jobs:
  scrape:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Set up Python
        uses: actions/setup-python@v5
        with:
          python-version: '3.11'

      - name: Install dependencies
        run: |
          python -m pip install --upgrade pip
          pip install -r requirements.txt
          python -m playwright install chromium

      - name: Run scraper
        env:
          BRIGHTDATA_USERNAME: ${{ secrets.BRIGHTDATA_USERNAME }}
          BRIGHTDATA_PASSWORD: ${{ secrets.BRIGHTDATA_PASSWORD }}
        run: python main.py
